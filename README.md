# Inframotionpro – inframotion GmbH Website

> Professional corporate website for **inframotion GmbH**, a German civil engineering and infrastructure services company.

Live Demo: https://fuatsevinc.github.io/inframotionpro/

---

## About the Project

This is a multi-page corporate website developed for inframotion GmbH, a company specializing in infrastructure, civil engineering, Elektrotechnik, and Saugbagger services. The site is built with HTML, CSS, and JavaScript and hosted via GitHub Pages.

---

| Page | Description |

## Getting Started

1. Clone the repository:
   git clone https://github.com/fuatsevinc/inframotionpro.git

2. Navigate into the project directory:
   cd inframotionpro

3. Open index.html in your browser or serve with a local server.
